# Security Policy

## Supported Versions

Currently, the `main` branch and the latest published PyPI release are actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 1.0  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a potential security issue in this project, we kindly ask you to notify us immediately. **Please do not create a public issue on GitHub.**

Instead, please send an email directly to [git@xerolux.de](mailto:git@xerolux.de) with details of the vulnerability.

We will make every effort to acknowledge your report quickly, and will provide an estimated timeline for the resolution. Once the issue is fixed, we will publish a new release containing the patch.
